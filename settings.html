PresetTrackerEnhanced (Beta2)

AI 메시지/스와이프 생성 시 사용된 프롬프트 정보(프리셋, 템플릿 등)를 자동으로 기록합니다.

채팅에서 캐릭터 이름을 클릭하면, 해당 응답 생성 시 사용된 모델명과 저장된 프롬프트 정보가 Toastr 팝업 알림으로 표시됩니다.

주의사항:
• 짧은 시간(1~2분) 내 설정 변경 및 응답 생성 시 기록이 부정확할 수 있습니다.
• 'Continue(계속 생성하기)' 기능 사용 시 기록된 정보를 잃어버릴 수 있습니다.

슬래시 커맨드:
• /pteCleanOrphanData: 삭제된 메시지의 불필요 데이터 정리 (필요시 수동 실행).
• /pteMigratePresetData: Beta1 버전 데이터 마이그레이션 (Beta2 이후에 설치한 사용자에게는 불필요).
